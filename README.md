# NLP Sentiment Classifier

A from-scratch NLP pipeline for sentiment analysis on the [IMDB 50K Movie Reviews](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews) dataset — built without high-level ML frameworks to understand the math behind it.

## Project Structure

```
src/
  clean.py        # Loads IMDB CSV and writes reviews to a flat text file
  tokenizer.py    # Custom tokenizer: cleans text, builds vocab, encodes/decodes
notebooks/
  vocabulary_analysis.ipynb  # Exploratory analysis of vocab, UNK rate, encodings
math-notes/
  cosine_similarity.md       # Handwritten derivation of cosine similarity
data/
  IMDB Dataset.csv           # Raw dataset (not tracked)
  sample_reviews.txt         # Extracted reviews (generated by clean.py)
```

## Getting Started

```bash
python3 -m venv venv
source venv/bin/activate
pip install pandas
python3 src/clean.py
```

## Key Concepts Covered

- Custom tokenizer with `<PAD>` / `<UNK>` tokens
- Vocabulary building from word frequency (top 10k words)
- Encode / decode pipeline for review text
- Cosine similarity — full mathematical derivation
